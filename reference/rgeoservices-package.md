# rgeoservices: A R Client for the IGN Geoservices API

A set of functions to query the API provided by the French National
Institute of Geographic and Forest Information (IGN) through its online
Geoplateform
<https://geoservices.ign.fr/documentation/services/services-geoplateforme>.
